projet

Lors de mon stage dans l'entreprise LS MARKETING, j'ai été chargé de m'occuper de certains de leurs clients.

Le client en question est un restaurateur de Blois, chez qui j'ai dû aller analyser son besoin pour lui proposer finalement un site CMS via Wordpress.

A l'aide de l'équipe, nous avons créé l'identité visuelle du restaurant, rentré tous les produits sur le site, laissant la possibilité au restaurateur de les modifier facilement.

De plus, le restaurant a la particularité de changer de produits régulièrement et l'agence est chargée de maintenir ses données à jour.

Front End, HTML CSS + CMS Wordpress - Woo Commerce
